\subsection{\-Detailed \-Description}
\-A function object that compares two real numbers using some relation. 

\-This comparison takes into account the limited precision of floating point values by using an arbitrarily chosen acceptable margin of error.

\begin{DoxyNote}{\-Note}
\-This is just an abstract class that doesn't correspond to any actual relation.
\end{DoxyNote}
