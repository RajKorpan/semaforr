Detailed Description

A function object that compares two real numbers using the equality relation.

This comparison takes into account the limited precision of floating point values by using an arbitrarily chosen acceptable margin of error.

Remarks:
Given two floating point numbers a and b, and an acceptable margin of error e, then a == b if and only if |a - b| <= e.

RealEquality::RealEquality ( double  epsilon = 0.0)

Construct a new predicate for comparing two real numbers using the equality relation.

Optionally provide an acceptable margin of error to be used in comparisons; if no acceptable margin of error is specified then it is assumed that there is no acceptable margin of error, that is that the margin of error is zero, which is no different than using the built-in comparison for floating point values.

  • epsilon is the acceptable margin of error

Treats the predicate as a function that takes two real numbers and returns the result of their comparison. For example, given two real numbers a and b and RealEquality f(e) where e is the acceptable margin of error, then f(a, b) == [fabs(a - b) <= e].
