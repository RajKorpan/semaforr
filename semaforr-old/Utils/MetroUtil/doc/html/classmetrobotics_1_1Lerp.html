Detailed Description

template<size_t dimension, size_t position = 0>
class metrobotics::Lerp< dimension, position >

A linear interpolation class for any N-dimensional vector space over the reals.

The purpose of this class is to provide an interface that allows the user to interpolate an N-dimensional point (represented by an N-dimensional vector) given one of its components (entries) such that there exists a pair of bounding data points in the set of all recorded data points. Put simply, this class can be used to interpolate (i.e. "fill in") the empty points on a graph by using the set of points that are already known together with one component (key) from those unknown points.

Template Parameters:
dimensionis the dimension of the vector space that we're working with; all points (vectors) in the set must be of the same dimension

Template Parameters:
positionis the position of the key (component) within the vector that is used for interpolation; a value of zero (the default value) corresponds to the first entry in the vector

Constructor & Destructor Documentation

template<size_t dimension, size_t position = 0>
metrobotics::Lerp< dimension, position >::Lerp ( ) [inline]

Construct a linear interpolation object with no data points (empty).

The linear interpolation object behaves like a container: it keeps a record of all inserted data points (vectors). Once all of the data points have been recorded, the process of interpolation is done automatically using the information that is already stored in the object itself.

Warning:
Because this linear interpolation class operates on an N-dimensional vector space over the real numbers it takes into account the limited precision that is inherent to representing real numbers on a computer. To that end, at object creation time, the value of RealVectorN<N>::marginOfError() will determine the acceptable margin of error for all of the operations on that object. Once the object has been created, subsequent changes to RealVectorN<N>::marginOfError() will have no effect on pre-existing linear interpolation objects. That said, make sure to set RealVectorN<N>::marginOfError() accordingly before creating a new linear interpolation object.

Insert a new data point (vector) into the set.

  • vec is the N-dimensional vector that represents the N-dimensional point to be added to the set of data points

Warning:
Inserting the same data point twice has no effect, but the entry at position must be unique so that for each possible value in that entry there can be only one correspending vector. That said, inserting a new vector containing a value at entry position that already exists in the set of points will cause the old vector containing that same value at entry position to be completely overwritten.

Interpolate an N-dimensional vector.

Interpolate an N-dimensional vector with respect to the key (component) at position using the data points that already exist in the set.

  • key is the entry (component) of the unknown vector (i.e. the vector that you're trying to find), which will be used to interpolate its remaining entries (components) with respect to the existing data points in the set
Returns:
an N-dimensional vector that contains the value of key at entry position, and whose remaining entries have been interpolated using the data points in the set
Exceptions:
std::domain_erroris thrown when the component used for interpolation is either lower than all the existing data points in the set or it is greater than all the existing data points in the set; in other words, the component would represent a vector that is outside the interval of recorded data points

value_type metrobotics::Lerp< dimension, position >::truncate ( const key_type &  key) const [inline]

Interpolate an N-dimensional vector with possible truncation.

Interpolate an N-dimensional vector with respect to the key (component) at position using the data points that already exist in the set. If the given key is outside the interval specified by the points, then the key is truncated (clamped) to the nearest endpoint, which is subsequently returned to the caller.

  • key is the entry (component) of the unknown vector (i.e. the vector that you're trying to find), which will be used to interpolate its remaining entries (components) with respect to the existing data points in the set; if key falls outside the interval specified by the points then it will be truncated to the nearest endpoint in the set
Returns:
an N-dimensional vector that contains the possibly truncated value of key at entry position, and whose remaining entries have been interpolated using the data points in the set
Exceptions:
std::logic_erroris thrown when the set of points is empty
